# Project Repository

This repository contains various tools and applications for visualization, deployment, and server management.

## Directory Structure

```
├── src/                    # Source code files
│   ├── *.py               # Python applications
│   ├── *.js               # JavaScript files
│   ├── *.html             # HTML files
│   └── servidor_descarga/ # Download server module
├── scripts/               # Shell scripts for deployment and management
├── docs/                  # Documentation and updates
├── config/                # Configuration files and services
└── README.md             # This file
```

## Components

### Source Code (`src/`)
- **Python Applications**: Various Python scripts for image creation, deployment, and server management
- **JavaScript**: Calculator and other interactive components
- **HTML**: Interactive web interfaces
- **Download Server**: Specialized server module

### Scripts (`scripts/`)
- Deployment scripts
- Application management scripts
- Installation utilities

### Documentation (`docs/`)
- Project updates and changelogs
- Deployment guides
- Feature documentation

### Configuration (`config/`)
- Service configuration files
- System service definitions

## Getting Started

1. Review the documentation in the `docs/` directory
2. Check the appropriate scripts in `scripts/` for deployment
3. Configure services using files in `config/`
4. Run applications from the `src/` directory

## Notes

- All image files are ignored by git (see `.gitignore`)
- System and configuration files are excluded from version control
- Backup files (*.bak, *.old) are automatically ignored
